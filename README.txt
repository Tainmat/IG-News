Tela de Notícias
Esta aplicação é uma tela de notícias desenvolvida em React JS com o gerenciador de pacotes VITE. Além disso, é utilizado a biblioteca de datas date-fns.

Funcionalidades
O usuário pode ver as notícias disponíveis na tela
O usuário pode criar uma nova notícia
O usuário pode reagir a uma notícia (curtir, compartilhar, comentar)

Instalação
Clone este repositório em seu computador
Abra um terminal e navegue até a pasta onde você clonou o repositório
Execute o comando npm install para instalar as dependências do projeto
Execute o comando npm run dev para rodar a aplicação localmente
Acesse a aplicação através do endereço http://localhost:3333

Tecnologias utilizadas
React JS
VITE
date-fns
